	
◆实验目的:
	学习ADC及光敏传感器的使用
	
◆硬件资源:

	1、请参照下表配置核心板上拨码开关，选择场景为“智慧农业”。
	 |EN | B | A | 
	 | 0 | 0 | 0 |
	  
	2、 P10 为光敏传感器的ADC采集引脚。


	
◆实验现象:
下载成功后，复位，tft屏幕上显示光敏传感器两端电压值。遮挡或强光照射传感器电压值有不同的变化。
	
◆参考资料：
    手册：《STC51实验指导书》。

◆说明：
AD功能是STC15自带的功能，STC15支持8 10位选择性AD电压采集。 
在此次函数实验里 我们用的是10位的AD采集  是用P1.0口来判断当期电压值，具体的AD寄存器设置 请看手册。
本程序3.3v作为基准电压。
