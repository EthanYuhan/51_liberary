	
实验目的:
	学习摄像头的使用
	
硬件资源:

	1、请参照下表配置核心板上拨码开关，选择场景为“工业互联网”。
	 |EN | B | A | 
	 | 0 | 1 | 1 |
	
	2、OV7670摄像头模块,连接关系为:
		OV7670模块 ----------- STM32实验平台
		 OV_D0~D7  -----------   P7 
		 OV_SCL    ------------  P10 
		 OV_SDA    ------------  P11 
		 OV_VSYNC  ------------  P33   
		 FIFO_RRST -----------   P13 
		 FIFO_OE   -----------   P14
		 FIFO_WRST ------------  P15
		 FIFO_WEN  ------------  P17 
		 FIFO_RCLK ------------  P16 
	
	3、本实验(汉字显示)需要SD卡配合，请插入SD卡（请勿带电插拔）。
	
	4、须将字库文件按手册拷贝到SD卡固定目录下(SYSTEM/GBK12.DZK)。

	
实验现象:
	
	本实验开机的时候先初始化OV7670，在初始化成功之后，就一直在屏幕显示OV7670拍到的内容。 
	(为尽量提高帧率可以设置单片机时钟为30MHZ)
	
注意事项:

	本实验需要使用杜比线将底板上面扩展IO的ETH_INT、OV_VSYNC插针短接在一起。（后续硬件升级不用杜邦线）

参考资料：
