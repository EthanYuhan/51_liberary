	
◆实验目的:
	学习PWM控制高亮LED亮度
	
◆硬件资源:

	1、请参照下表配置核心板上拨码开关，选择场景为“智慧农业”。
	 |EN | B | A | 
	 | 0 | 0 | 0 |
	  
	2、PWM7的P17控制高亮LED亮度。
	
	3、单片机时钟设置为24M HZ。
	


	
◆实验现象:
	本实验通过代码控制底板上的高亮LED亮度，高亮led由暗到亮,再由亮到暗,再由暗到亮,依次循环。 
	
◆参考资料：
    手册：《STC51实验指导书》。

基于STC官方15W4K-6路PWM例程完成。
