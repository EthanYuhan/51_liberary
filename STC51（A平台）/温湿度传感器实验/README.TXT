	
实验目的:
	学习DH11温度传感器的使用
	
硬件资源:

1、请参照下表配置核心板上拨码开关，选择场景为“智慧农业”。
	 |EN | B | A | 
	 | 0 | 0 | 0 |
			
2、DHT11检测引脚为P55。

3、本实验需要 TFT 液晶配合，请确保“TFT背光”跳线帽已插好。
   
	
实验现象:
	
	屏幕显示温湿度信息。
