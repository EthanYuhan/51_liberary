【 ！】功能简介：

非特定人语音识别。


【*】注意事项：
1、请参照下表配置核心板上拨码开关，选择场景为“智慧农业”。
	 |EN | B | A | 
	 | 0 | 0 | 0 |

2、语音识别过程中请靠近咪头发出指令。


【*】 引脚分配
//P11<----> RST    
接地<----> WR(SPIS)   
不接<----> IRQ   采用轮训式
P41<----> MISO				
P40<----> MOSI				
P43<----> SCK				
//P52<---->cs(NSS)  

P12<---->cs(NSS) 
P13<----> RST   

											
【*】 实验现象

在语音识别例程中，通过对语音识别模块的咪头说出“流水灯”“闪烁”“全灭”
“开灯”四种可识别的指令，语音识别模块完成对非特定人语音识别后可完成相应的指令。
