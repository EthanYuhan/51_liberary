	
实验目的:
	学习vs1053音乐播放器芯片的使用
	
硬件资源:
	1、请参照下表配置核心板上拨码开关，选择场景为“智能音箱”。
	 |EN | B | A | 
	 | 0 | 1 | 0 |


	
实验现象:
	
	程序流程：根据文件系统打开tf卡里存好的音乐文件 并直接进行播放
	           音乐的音量都是已经设置好的  
			   播放完整首歌后 程序会停止在while里不动
			   显示功能方面为全英文显示 没有加汉字功能

	 音乐路径：音乐播放路径 为 	 /music/泡沫.mp3
			   如果有其他歌曲要换 改这里就可以

	 tf卡要求：小于4g  可以格式化成fat32文件系统形式  
	           将资料里准备好的music文件夹直接放入已经格式好的tf卡根目录下
			   程序会直接打开music文件下已经存好的音乐文件 泡沫  然后进行播放

	 文件系统：采用fatfs 官网http://elm-chan.org/fsw/ff/00index_p.html
	           小型文件系统 Petit FatFs Module
			   目前文件系统版本为 June 10, 2014
