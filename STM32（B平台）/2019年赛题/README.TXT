	
实验目的:
基于TFTLCD显示屏例程完成2019年赛题题目。
	
硬件资源:
	1,LED0(连接在PA15)
	2,串口1(波特率:115200,PA9/PA10连接在板载USB转串口芯片CH340上面)
	3,实验平台自带3.5 寸TFTLCD模块(通过FSMC驱动 ) 
	
实验现象:
	
	下载成功后，，按下复位之后，就可以看到LCD模块不停的显示一些信息并不断切换底色。同时该实验会显示LCD驱
	动器的ID，并且会在串口打印（按复位一次，打印一次）。
