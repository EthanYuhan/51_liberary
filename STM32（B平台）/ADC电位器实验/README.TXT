	
实验目的:
	学习STM32内部ADC的使用
	
硬件资源:

	2,串口1(波特率:115200,PA9/PA10连接在板载USB转串口芯片CH340上面)
	3,实验平台自带3.5 寸TFTLCD模块(通过FSMC驱动 ) 
	4,ADC(STM32内部ADC1,通道8,即:ADC1_CH8)
	
实验现象:
	本实验通过STM32内部ADC1读取通道1（PB0）上面的电压，该电压可通过电位器调节大小。	
	在LCD模块上面显示ADC转换值以及换算成电压后的电压值。

	
注意事项:
	1,4.3寸和7寸屏需要比较大电流,请用外部电源适配器(推荐外接12V 1A电源).
	2,本例程在LCD_Init函数里面(在ILI93xx.c),用到了printf,如果不初始化串口1,将导致液晶无法显示!!
	3,ADC的最大输入电压是3.3V,请不要超过这个值.
	4,ADC的参考电压默认 连接在VDDA上面(3.3V),所以默认参考电压是3.3V
	5,请参照下表配置核心板上拨码开关，选择场景为“智能小车”。
	 |EN | B | A | 
	 | 0 | 0 | 1 |
