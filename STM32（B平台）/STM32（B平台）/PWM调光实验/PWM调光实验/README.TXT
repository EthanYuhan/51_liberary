
	
实验目的:
	学习定时器的PWM功能
	
硬件资源:
	1,D3高亮led(连接在PB7)
	2,定时器3(TIM3),使用TIM3的通道2(CH2),开启部分重映射,将TIM3_CH2输出到PB7.
	从而实现PWM输出控制D3高亮led亮度.
	
实验现象:
	本实验,D3高亮led由暗到亮,再由亮到暗,再由暗到亮,依次循环.
	
注意事项:
	请在参照下表配置核心板上拨码开关，选择场景为“智慧农业”。
	 |EN | B | A | 
	 | 0 | 0 | 0 |
